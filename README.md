# Backup Management System

## 📌 Description
A Java Swing desktop application integrated with MySQL that allows users to manage backup records with authentication and CRUD operations.

## 🚀 Features
- User Login System
- Add Backup Records
- View Backup Records (JTable)
- Delete Backup Records
- MySQL Database Integration

## 🛠️ Tech Stack
- Java (Swing)
- MySQL
- JDBC
- NetBeans IDE

## 🧠 Learning Outcomes
- GUI development in Java
- Database connectivity using JDBC
- SQL operations (CRUD)
- Event-driven programming
